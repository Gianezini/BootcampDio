Criado um bando de dados para ecommerce, com as seguintes tabelas:

1. Cliente
2. Estoque
3. Fornecedor
4. LocalEstoque
5. Pagamento
6. Pedido
7. Produto
8. ProdutoFornecedor
9. ProdutoVendedor
10. Vendedor

As tabelas possuem relacionamento entre chaves primária e estrangeira, bem como algumas constraints
para padronizar algumas ações.

Nesse repositório estão os scripts para criação do banco ecommerce, população desse banco, e algumas queries respondendo algumas perguntas de negócios, relacionadas as informaçõe dessa base de dados.
